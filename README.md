cVigenere
==================

## Installation

```
$ git clone git@github.com:jnreynoso/cvigenere.git
```
### Pre-requisites

    # Java   (Windows or Linux)
    # Csharp (Windows C# Compiler or Mono for Linux enviroments) 
    
### Install Mono Ubuntu

    # sudo apt-get install mono-xsp2 mono-xsp2-base
   
## Usage

`Java`
```
$ javac Crypt.java
$ java Crypt
```
`Csharp` using`mono`
```
$ gmcs Crypt.cs
$ mono Crypt.exe
```
